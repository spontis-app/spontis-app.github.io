About SPONTIS

SPONTIS is a lightweight, zero-login guide to what’s happening in Bergen right now. We collect public schedules from trusted organisers, normalise timezones, validate the schema and remove past events automatically so the list stays fresh.

Everything runs as open data on GitHub Pages. Filters are fast, mobile-friendly and tuned for dark mode. Tap any listing to jump to the original organiser for tickets or details.
